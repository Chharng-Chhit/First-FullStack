# Vue Task Frontend

This is the frontend for the 6-hour Vue.js + Laravel full stack training.

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

## Main Concepts Used

- `ref()` for reactive data
- `computed()` for derived values
- `onMounted()` to load data when page opens
- `v-model` for form binding
- `v-for` for task list rendering
- `v-if` for conditional button display
- `@click` and `@submit.prevent` for events
- Axios for API requests
