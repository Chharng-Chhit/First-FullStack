# 6H Teaching Flow for This Code

## 0:00 - 0:30 Full Stack Concept
Explain frontend, backend, API, database, JSON, CRUD.

## 0:30 - 1:15 Laravel Setup
Show Laravel folder structure, `.env`, route, controller, model, migration.

## 1:15 - 2:00 Laravel CRUD API
Create `tasks` table, `Task` model, `TaskController`, and API route.

## 2:00 - 2:15 Break

## 2:15 - 3:00 API Testing
Test API with Postman/Insomnia.

## 3:00 - 4:00 Vue Setup
Show Vue project structure, `ref`, `reactive`, `onMounted`, `v-model`, `v-for`, events.

## 4:00 - 5:00 Connect Vue to Laravel
Use Axios to call API and display tasks.

## 5:00 - 5:45 Complete CRUD
Create, update status, delete task.

## 5:45 - 6:00 Review
Ask students to create Note Manager using the same structure.
